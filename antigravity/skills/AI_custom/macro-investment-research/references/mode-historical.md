# Mode 3：Historical Cycle & Policy Transmission（歷史週期與政策傳導對比）

## 模式定位

將當前的總經結構條件與**指定的歷史週期**進行深度參數對比，提取歷史教訓並識別當前週期與歷史的關鍵差異。此模式強調「歷史不會重演，但會押韻」——目標是找出**韻腳在哪裡、走音在哪裡**。

> [!IMPORTANT]
> 此模式不是簡單的事件年表對比，而是**結構性參數對比**——比較兩個時代的貨幣工具、實質需求來源、市場槓桿結構與政策回應空間。

---

## 工作流程

### 步驟 1：歷史錨定與時代選擇
1. 確認使用者指定的歷史參照期（如：2000 達康泡沫、2008 次貸危機、1970s 停滯性通膨、2011 QE2）。
2. 若使用者未指定，根據當前總經主題**主動建議** 1-2 個最相關的歷史類比，並說明選擇理由。
3. 定義「當前週期」與「歷史週期」的時間邊界。

### 步驟 2：PESTEL 跨時代結構對比
- 引用 `core-toolkit.md` → 工具 1（PESTEL）
- **Historical 聚焦**：以 PESTEL 框架建立**並排對比表**

| PESTEL 面向 | 歷史週期 [年份] | 當前週期 [年份] | 結構差異 | 對資產定價的影響差異 |
|:------------|:----------------|:----------------|:---------|:--------------------|

- 每個面向必須標記：**相似**、**部分相似**、**根本不同**。

### 步驟 3：三維度深度對比矩陣
建立以下三個維度的系統性對比：

#### 維度 1：貨幣工具對比
| 比較項目 | 歷史週期 | 當前週期 | 差異解讀 |
|:---------|:---------|:---------|:---------|
| 主要政策工具 | （如：傳統降息） | （如：QE + 前瞻指引） | |
| 政策利率水位 | | | |
| 央行資產負債表規模 | | | |
| 政策空間（可用工具餘量） | | | |
| 貨幣政策傳導效率 | | | |

#### 維度 2：實質需求對比
| 比較項目 | 歷史週期 | 當前週期 | 差異解讀 |
|:---------|:---------|:---------|:---------|
| 全球資本支出主力 | （如：中國基建） | （如：AI 資料中心） | |
| 消費動能來源 | | | |
| 全球化 vs 去全球化 | | | |
| 人口結構趨勢 | | | |
| 技術週期階段 | | | |

#### 維度 3：市場結構對比
| 比較項目 | 歷史週期 | 當前週期 | 差異解讀 |
|:---------|:---------|:---------|:---------|
| 槓桿載體 | （如：CDO/MBS） | （如：私募信貸/CLO） | |
| 散戶 vs 法人參與比例 | | | |
| 被動投資佔比 | | | |

### 步驟 4：傳導路徑對比圖
- 引用 `core-toolkit.md` → 工具 2（Mermaid 傳導圖）
- **Historical 聚焦**：**並排兩張 Mermaid 圖**，分別展示歷史與當前的傳導路徑
- 以不同顏色/標記凸顯兩者的結構差異節點

### 步驟 5：歷史三部曲映射
- 引用 `core-toolkit.md` → 工具 3（三部曲時間軸）
- **Historical 聚焦**：
  1. 先建立歷史事件的三部曲回顧（已知結果）
  2. 再建立當前週期的三部曲推演（預測）
  3. 對比兩者在每個階段的相似與差異

### 步驟 6：歷史教訓與資產啟示
- 引用 `core-toolkit.md` → 工具 4（美林投資時鐘）
- **Historical 聚焦**：回測歷史同週期定位下的實際資產表現
- 產出：歷史類比的投資啟示（適用的部分 + 不適用的部分）

### 步驟 7：主動風險偵測
- 引用 `proactive-triggers.md`
- 特別關注歷史週期中「沒發生但當前可能發生」的風險
