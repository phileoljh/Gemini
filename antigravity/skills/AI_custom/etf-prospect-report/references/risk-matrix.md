# 風險矩陣、投資評等與信心標籤 (Risk Matrix)

> 本文件定義 ETF 分析中的風險等級判定標準、投資評等條件與資料信心標籤。所有判定必須嚴格依照以下條件執行。

---

## 一、風險等級標準

### 判定規則
綜合評估 ETF 的集中度、費用、追蹤、流動性與策略風險後，依照以下條件給出風險等級。**風險等級後必須附上 2~3 句說明**，引用具體數據解釋為什麼歸類為此等級。

| 風險等級 | 條件 | 適合投資人 |
|---|---|---|
| 🟢 低風險 | 市值型大盤指數 ETF + 費用率 < 0.10% + AUM > $50B + 日均量 > 10M + 前十大佔比 < 30% | 保守型、長期存股族 |
| 🟡 中低風險 | 主流指數 / 高股息 ETF + 費用率 < 0.30% + AUM > $5B + 流動性充足 + 追蹤誤差小 | 穩健型 |
| 🟠 中等風險 | 產業型 / 主題型 ETF + 費用率 < 0.60% + 集中度較高 + AUM > $1B | 平衡型 |
| 🔴 中高風險 | 高度集中或新興主題 ETF + 費用率 > 0.60% + AUM < $1B + 或單一國家 ETF | 積極型 |
| ⚫ 高風險 | 槓桿 / 反向 ETF + 規模極小 + 流動性差 + 不適合長期持有 | 短線交易者，需嚴格停損 |

---

## 二、風險評估指標（完整 7 項）

報告的風險評估表格必須涵蓋以下 7 項指標：

| 指標 | 數值格式 | 判讀標準 |
|---|---|---|
| 前十大集中度 | XX.X% | < 30% 分散 / 30~50% 適中 / > 50% 集中 |
| 費用率 vs 同類 | X.XX% vs X.XX% | 低 / 適中 / 偏高（同類型比較） |
| 追蹤誤差 | X.XX% | < 0.10% 優良 / 0.10~0.50% 正常 / > 0.50% 偏大 |
| 基金規模 AUM | $XX.XB | > $10B 充足 / $1B~$10B 適中 / < $1B 偏小 / < $100M 清算風險 |
| 日均成交量 | XX.XM | > 1M 充足 / 100K~1M 適中 / < 100K 偏低 |
| NAV 折溢價 | ±X.XX% | < ±0.5% 正常 / ±0.5~1% 留意 / > ±1% 偏高 |
| 資金流向 | 淨流入/流出 $X.XB | 淨流入 正面 / 持平 中性 / 淨流出 留意 |

---

## 三、投資評等標準

### 判定規則
必須嚴格依照以下條件判定，同時納入風險等級。**評等後必須附上 2~3 句完整的理由說明**，解釋為什麼給出這個評等，引用前面分析中的具體數據（含風險等級）。

| 評等 | 條件 |
|---|---|
| ⭐⭐⭐⭐⭐ 強力推薦 | 追蹤誤差極小 + 費用率同類最低 + 規模大流動性強 + 配息穩定 + 績效優於同類 + 風險 🟢🟡 |
| ⭐⭐⭐⭐ 建議買入 | 追蹤準確 + 費用率合理 + 規模充足 + 配息正常 + 風險 🟢🟡🟠 |
| ⭐⭐⭐ 中立觀望 | 各項指標中等 + 或成立時間較短數據不足 + 或風險 🟠 以上需觀察 |
| ⭐⭐ 建議減碼 | 費用率偏高 + 追蹤誤差大 + 流動性不足 + 資金淨流出 + 績效落後同類 |
| ⭐ 建議避開 | 費用率過高 + 規模極小有清算風險 + 追蹤嚴重偏離 + 持續大量資金流出 + 風險 🔴⚫ |

---

## 四、資料信心標籤

每次分析結尾必須標註信心程度，依據資料的時效性與來源完整度評定：

| 信心等級 | 條件 |
|---|---|
| 🟢 **高信心** | 所有關鍵數據皆為當日/最近交易日，且來自權威來源（基金公司官網、ETF.com、Yahoo Finance 等），多個來源交叉驗證一致 |
| 🟡 **中信心** | 部分數據為近幾日內資料，或部分欄位僅有單一來源，或少數次要數據缺漏 |
| 🔴 **低信心** | 關鍵數據缺漏較多（如費用率、AUM 未取得）、資料時效超過一週、或來源可信度不足 |

### 信心標籤多語言對照
- 繁體中文：🟢 高信心 / 🟡 中信心 / 🔴 低信心
- English：🟢 High Confidence / 🟡 Medium Confidence / 🔴 Low Confidence
- 日本語：🟢 高い信頼度 / 🟡 中程度の信頼度 / 🔴 低い信頼度
